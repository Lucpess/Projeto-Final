\doxysubsection{Detailed Description}
Classe que implementa o jogo de \doxylink{class_reversi}{Reversi}. 

A classe {\ttfamily \doxylink{class_reversi}{Reversi}} define os métodos e atributos necessários para gerenciar o estado do jogo, verificar jogadas válidas e atualizar o tabuleiro. Ela herda da classe {\ttfamily \doxylink{class_jogos}{Jogos}} e sobrecarrega alguns dos seus métodos para adaptar à lógica do \doxylink{class_reversi}{Reversi}.
