\doxysubsection{Detailed Description}
Classe que representa o tabuleiro de um jogo e suas operações. 

A classe {\ttfamily \doxylink{class_tabuleiro}{Tabuleiro}} é usada para gerenciar o estado do tabuleiro, incluindo a inicialização, modificação e impressão do mesmo.
