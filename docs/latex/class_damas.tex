\doxysubsection{Detailed Description}
Representa o jogo de \doxylink{class_damas}{Damas}. 

A classe {\ttfamily \doxylink{class_damas}{Damas}} herda de {\ttfamily \doxylink{class_jogos}{Jogos}} e implementa a lógica específica do jogo de \doxylink{class_damas}{Damas}. Inclui métodos para inicializar o jogo, gerenciar os turnos dos jogadores, movimentar peças, verificar jogadas válidas e determinar se houve uma vitória.
