\doxysubsection{Detailed Description}
Classe que implementa o jogo \doxylink{class_lig4}{Lig4} (Conecte 4). 

A classe \doxylink{class_lig4}{Lig4} herda de \doxylink{class_jogos}{Jogos} e fornece toda a funcionalidade necessária para iniciar e gerenciar uma partida de \doxylink{class_lig4}{Lig4}, incluindo verificação de jogadas, controle de turnos e detecção de vitória ou empate.

Verifica se o jogo chegou ao fim após a última jogada. 

Esta função é chamada após cada jogada para determinar se a partida terminou. A função verifica se a última peça colocada resultou em uma sequência de 4 peças consecutivas, declarando vitória para o jogador correspondente. Além disso, verifica se o tabuleiro está cheio, indicando um empate caso nenhuma outra jogada possa ser feita.
