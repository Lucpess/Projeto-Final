Detailed Description

Classe base para jogos que utilizam um tabuleiro.

A classe Jogos fornece a interface e funcionalidades comuns para todos os jogos que utilizam um tabuleiro. Inclui métodos para iniciar o jogo, imprimir o tabuleiro, e gerenciar o turno dos jogadores.
